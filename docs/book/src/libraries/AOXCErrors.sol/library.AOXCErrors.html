Title: AOXCErrors

Author: AOXC Core Engineering

Centralized library for all AOXC Protocol custom errors.

Optimized for gas efficiency and Akdeniz V2 26-channel forensic logging. Categories: 1xxx (Access), 2xxx (Finance), 3xxx (Forensic), 4xxx (Assets), 5xxx (Gov), 6xxx (Infra).
